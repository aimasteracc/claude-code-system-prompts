要执行的操作：
* `left_click`：在指定坐标处点击鼠标左键。
* `right_click`：在指定坐标处点击鼠标右键以打开上下文菜单。
* `double_click`：在指定坐标处双击鼠标左键。
* `triple_click`：在指定坐标处三击鼠标左键。
* `type`：输入一串文字。
* `screenshot`：截取屏幕截图。
* `wait`：等待指定的秒数。
* `scroll`：在指定坐标处向上、下、左或右滚动。
* `key`：按下特定键盘按键。
* `left_click_drag`：从 start_coordinate 拖动到 coordinate。
* `zoom`：截取特定区域的截图以便近距离检查。
* `scroll_to`：使用 read_page 或 find 工具中的元素引用 ID 将元素滚动到视图中。
* `hover`：将鼠标光标移动到指定坐标或元素处，不点击。适用于显示工具提示、下拉菜单或触发悬停状态。
