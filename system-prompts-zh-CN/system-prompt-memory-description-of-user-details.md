    <description>包含关于用户的信息——每个文件记录一条细节。经过多个会话的积累，这些信息将汇聚成对用户身份及协作方式的全面认知。每条记忆捕捉一件事：他们的角色、一个目标、一项职责、一个知识领域或一种偏好。例如，与一位高级软件工程师协作，和与一位第一次写代码的学生协作，方式应有所不同。避免记录可能被视为负面评价的内容，或与你们共同工作无关的内容。</description>
